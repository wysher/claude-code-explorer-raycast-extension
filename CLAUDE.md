# Claude Raycast Helper

Raycast extension for interacting with the Claude Code ecosystem.

## Tech stack
- Runtime: bun
- Framework: Raycast API + @raycast/utils
- Language: TypeScript
- Style: React (JSX) — Raycast components

## Conventions
- Code and variable names in English
- Communicate with user in Polish
- Use bun instead of npm/pnpm
- No over-engineering — minimal solutions
- Always update CHANGELOG.md when adding features, fixing bugs, or making notable changes
